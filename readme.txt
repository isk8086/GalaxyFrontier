  某祭りに出すことを決めた。なので、作る。
  一週間で。
  んで、できた。


・Copyright の「2003」ってなに？

  一応前作に当たる「RAPID HELLO」をリリースした年です。
  ８年前ですか。進歩ないですね。


・動かし方

  このフォルダごと HDD とかにコピーしてください。
  そんで、中の「shooting.exe」を起動してください。
  でも、添付のバッチファイルから起動したほうがきっといいです。


・ルール

  敵は本気で潰しにかかりやがってくるので、あなたはいつも通り本気で敵をつぶしに
かかりやがってください。
  敵の本気具合があなたの本気具合に勝ってしまったら、あなたは死にます。
  なんかできそこないの英日翻訳みたいな文章ですが、意味を汲み取って（適宜読み替
えて）下さるとこちらとしては幸いです。


・操作説明

  ＜タイトル＞
  Zキー   ：決定

　＜メニュー＞
  項目選択：カーソル上下
  Zキー   ：決定
  Xキー   ：キャンセル

  ＜ゲーム中＞
  自機移動：カーソルキー
  Zキー   ：ショット
  Pキー   ：ポーズ

  ジョイパッドに対応しています。

  ESC を押すと RAPID HEL(ry を終了します。


・きちのもんだい

  いろいろひどい。
  

＜最後に＞

・謝辞

  RAPID HELLO 1-2-3! は C言語 で書かれています(GCC ver. 3.4.4)。

  画面の出力には Simple DirectMedia Layer を利用しています。
    Simple DirectMedia Layer
    http://www.libsdl.org/

  BGM と SE の出力に SDL_mixer と Ogg Vorbis CODEC を利用しています。
    SDL_mixer 1.2
    http://www.libsdl.org/projects/SDL_mixer/
    Vorbis.com
    http://www.vorbis.com/

  SDL ラッピングライブラリ libSDL_wheel を利用しています。
    SDL_wheel 0.12
    http://isshiki.la.coocan.jp/SDL_page/SDL_wheel/
